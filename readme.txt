== Description ==

Awesome Gallery Widget Plugin for WordPress provides you a very attractive photo gallery widget with a great lightbox effect.

**Features**

* Can have a multiple widget on multiple section of the website
* Can add multiple images to rotate as a gallery.
* Can set a lightbox popup effect on gallery photo.
* Can re-order gallery images via simply Drag & Drop method.
* Can set a number of columns under each gallery.

**More for info & Demo**

<a href="https://desirepress.com/plugin/awesome-gallery-widget/" target="_blank">Click Here</a>


== Installation ==

* Upload the plugin folder to the /wp-content/plugins/ folder of your WordPress installation.  
* Activate the plugin.   
* You will see a widget on WordPress Widget page to initiate that..  

Enjoy it......
